# 😷 Face Mask Detection using YOLOv8 with Explainable AI

## 📖 Project Overview
This project focuses on detecting whether individuals are wearing face masks using the **YOLOv8 object detection model**.  
It also integrates **Explainable AI (XAI)** techniques such as **SHAP, LIME, and Grad-CAM** to interpret model predictions and improve transparency.

---

## 🎯 Objectives
- Detect face mask usage in public environments  
- Improve model interpretability using XAI  
- Enhance trust and understanding of AI decisions  

---

## 🧠 Technologies Used
- Python  
- YOLOv8 (Ultralytics)  
- OpenCV  
- NumPy  
- SHAP (SHapley Additive exPlanations)  
- LIME (Local Interpretable Model-Agnostic Explanations)  
- Grad-CAM (Gradient-weighted Class Activation Mapping)  
- Google Colab  

---

## ⚙️ Methodology
1. Dataset collection and preprocessing  
2. Training YOLOv8 model for mask detection  
3. Model evaluation and predictions  
4. Applying XAI techniques:
   - **SHAP** → Feature importance analysis  
   - **LIME** → Local prediction explanations  
   - **Grad-CAM** → Visual heatmaps for model attention  

---

## 📊 Results
- Accurate detection of masked and unmasked faces  
- Visualization of decision-making using heatmaps and feature importance
